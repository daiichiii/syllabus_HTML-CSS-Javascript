ヴェノム

人体実験で死者を出しているという財団の真相を探る敏腕記者エディ。そのさなかに“シンビオート”と呼ばれる地球外生命体に触れた彼は、それに寄生されてしまう。やがて両者が一体化した“ヴェノム”は、グロテスクな風貌で容赦なく人々に襲いかかっていく。

アイアンマン

天才発明家にして軍事企業経営者でもあるトニーは、アフガンでテロリストに拉致される。彼らから兵器開発を強いられるが、密かにパワードスーツを作り上げて脱出。それを機にトニーはパワードスーツを装着した正義のヒーロー“アイアンマン”として活躍する。
